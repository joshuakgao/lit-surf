# Construction Monitoring — Index

Research on automated construction progress monitoring and change detection using 3D point clouds and Building Information Models. Focus on detecting as-built vs. as-planned deviations, quantifying construction progress, and enabling objective project management through multi-temporal sensor data analysis. Key themes: geometric change detection, semantic interpretation, uncertainty quantification, and integration with BIM workflows.
