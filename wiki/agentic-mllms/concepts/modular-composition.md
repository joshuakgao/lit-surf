## Definition

Modular composition in vision is the decomposition of complex visual reasoning tasks into chains of simpler, reusable operations—each performing a specific visual or linguistic capability (object detection, property verification, spatial reasoning, counting, depth estimation, semantic reasoning). The output of one module serves as input to the next, creating explicit step-by-step reasoning without end-to-end training.

## Origins

Pioneered by Neural Module Networks (Andreas et al., 2015), which proposed decomposing visual reasoning into atomic perceptual units arranged dynamically. Classical neuro-symbolic systems followed this principle. ViperGPT revives and modernizes modular composition by leveraging LLMs to synthesize the composition dynamically for each query.

## Key variants / extensions

- **Manually composed systems**: Hard-coded module chains for specific tasks; does not generalize
- **Learned layouts**: End-to-end training of program structure and module parameters jointly; computationally expensive and often fails to learn true modularity
- **LLM-synthesized composition**: Program generated by an LLM given an API specification; achieves zero-shot generalization without task-specific training
- **Differentiable module networks**: Trainable but differentiable module compositions; intermediate approach between hard-coded and learned
- **Sparse composition**: Composing only a small subset of available modules; improves efficiency

## Advantages and challenges

**Advantages:**
- Interpretability: each step is explicit and inspectable
- Modularity: improvements in any module directly improve downstream performance
- Generalization: composition learned from scratch via LLM priors, not memorized from task distribution
- Debuggability: failures can be traced to specific module or step

**Challenges:**
- Error propagation: mistakes in early steps affect downstream results; no error recovery
- API completeness: system quality depends on availability of the right primitives
- Complexity: some tasks may require very long or intricate compositions
