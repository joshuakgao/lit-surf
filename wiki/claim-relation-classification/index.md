# Claim Relation Classification — Index

This topic covers work on identifying and classifying how scientific claims from different papers relate to one another. The core task is to predict the epistemic relation (support, extend, qualify, refute, or background reference) between a cited claim and a citing claim, using the citation context and claim text as evidence. This enables fine-grained analysis of how scientific ideas propagate, evolve, and interact across the literature.
