## Definition

A cognitive architecture is a theoretical framework describing the foundational mental faculties or capabilities that comprise intelligent reasoning. In the context of video reasoning, it organizes core visual reasoning capabilities into distinct, measurable dimensions.

## Origins

Rooted in cognitive science and philosophy of mind (Newell & Simon, 1972; Anderson, 2007). VBVR (2026) applies these principles to video reasoning, mapping the five foundational human cognitive faculties to video reasoning tasks.

## Key variants / extensions

**Five cognitive capabilities (VBVR framework):**
- **Abstraction**: generalizing patterns, categorizing concepts, extracting commonalities
- **Knowledge**: semantic understanding, real-world reasoning, commonsense inference
- **Spatiality**: spatial relationships, layout reasoning, object localization and orientation
- **Perception**: fine-grained visual discrimination, sensory detail, color/texture reasoning
- **Transformation**: object motion, state changes, dynamics, temporal sequences

Each capability can be further decomposed into sub-skills (e.g., Spatiality → relative position, containment, symmetry).
